E-LEARNING PLATFORM — SETUP SUMMARY
====================================
Last updated: 2026-06-15

PREREQUISITES
- Node.js (v20+)
- Supabase free tier (PostgreSQL via Supabase JS client)
- npm

TECH STACK
- Next.js 16 (App Router, webpack — use `npm run dev` which passes `--webpack`)
- Tailwind CSS v4 + shadcn/ui (radix-nova)
- Supabase for auth + database (service-role key, no direct postgres)
- framer-motion for animations
- lucide-react for icons

ENVIRONMENT (.env) — fill these 3 + optional payment vars:
  NEXT_PUBLIC_SUPABASE_URL          — Project Settings → API → Project URL
  NEXT_PUBLIC_SUPABASE_ANON_KEY     — Project Settings → API → anon public key
  SUPABASE_SERVICE_ROLE_KEY         — Project Settings → API → service_role key
  BANK_NAME, BANK_ACCOUNT_NAME, BANK_ACCOUNT_NUMBER, BANK_BRANCH_CODE, PAYMENT_REFERENCE_PREFIX — optional, for EFT payments
  NVIDIA_API_KEY                    — optional, for AI tutor

SETUP (using Supabase SQL Editor)
  1. Go to https://supabase.com/dashboard/project/nahrsxuftwrdpcldnurw
  2. Open the SQL Editor
  3. Open prisma/migration.sql → copy all → paste in SQL Editor → Run (creates all tables)
  4. npm run seed:auth     (creates demo users)
  5. npm run seed:courses  (creates 20 courses)
  6. npm run seed:plans    (creates pricing plans)
  7. npm run dev

WHAT WAS BUILT (2026-06-14)

  Phase 1 — Foundation:
  - Next.js 16 + TypeScript + Tailwind v4 + App Router + webpack
  - Prisma 7 schema (17 models), PrismaClient with PrismaPg adapter
  - Auth.js v5 (Credentials, JWT, role-based: STUDENT/INSTRUCTOR/ADMIN)
  - shadcn/ui (15 components: button, input, label, card, dialog, dropdown-menu,
    avatar, badge, select, separator, sheet, sonner, checkbox, textarea, radio-group)
  - Role-based middleware (/dashboard, /instructor, /admin)
  - Pages: Home, Login, Register

  Phase 2 — Courses & Learning:
  - Course CRUD with instructor management pages
  - Module & lesson CRUD on edit course page
  - Public course catalog with category filter chips + average rating display
  - Course detail page with module/lesson outline + enrollment + review section
  - Lesson viewer with prev/next nav, video playback, completion tracking
  - Student enrollment (free courses) with auto-progress tracking
  - Student dashboard with enrolled courses + progress bars
  - Quiz engine: instructors create quizzes with MC/TF/SA questions + options
  - Auto-grading on submission, results page with score %
  - Previous attempt history shown on quiz page
  - Reviews & ratings: students rate 1-5 stars with optional comment
  - Upsert reviews (one per student), auto-updates course rating
  - AI Tutor powered by NVIDIA NIM (Llama 3.1 8B) with RAG on course content
  - Slide-out chat panel on lesson viewer page
  - Free course import from Khan Academy API
  - Instructor sidebar layout (Overview, My Courses, Import)
  - Seed script: 20 categories + 3 demo users
  - Instructor dashboard with stats (courses, students, revenue)
  - Supabase JS client for all DB access (service_role key, no direct postgres)
